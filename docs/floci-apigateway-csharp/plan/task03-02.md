## 目的

`TodoApiService` を実装し `POST /todos` / `GET /todos/{id}` を呼び出す。`HttpErrorResponse` を `ApiErrorResponse` 形式に整形し再 throw。`status === 0` (CORS / network) を区別する。

## 実装ステップ (TDD)

### RED
1. `frontend/src/app/services/todo-api.service.spec.ts` を新規作成 (FAIL):
   - **UT-3**: `create({title:'a'})` → `apiBaseUrl + /todos` に POST、Content-Type 検証
   - **UT-4**: `get('id1')` → `apiBaseUrl + /todos/id1` に GET
   - **UT-5**: 400 `{errors:['title is required']}` → `catchError` で `ApiErrorResponse` 型で再 throw

   - **UT-7**: `status===0` (CORS/network) → network エラー識別子付きで再 throw

### GREEN
2. `frontend/src/app/services/todo-api.service.ts` を実装:
   ```typescript
   @Injectable({ providedIn:'root' })
   export class TodoApiService {
     constructor(private http: HttpClient, private cfg: ConfigService) {}
     create(req: TodoCreateRequest): Observable<Todo> {
       return this.http.post<Todo>(`${this.cfg.apiBaseUrl}/todos`, req).pipe(
         catchError(e => throwError(() => this.mapError(e))));
     }
     get(id: string): Observable<Todo> {
       return this.http.get<Todo>(`${this.cfg.apiBaseUrl}/todos/${encodeURIComponent(id)}`).pipe(
         catchError(e => throwError(() => this.mapError(e))));
     }
     private mapError(e: HttpErrorResponse): ApiErrorResponse & { status: number } {
       if (e.status === 0) return { error: 'network', status: 0 };
       const body = e.error ?? {};
       return { ...body, status: e.status };
     }
   }
   ```
3. `npm run test:unit` で **GREEN**
