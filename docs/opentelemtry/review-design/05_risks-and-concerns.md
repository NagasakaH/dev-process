# リスク・懸念事項

## 1. 概要

設計上のリスク、調査結果で指摘されたリスクへの対応確認、見落としているリスクの特定を行います。

## 2. リスク対応確認

### 2.1 調査で特定されたリスクへの対応

| リスク（06_risks-and-constraints.md） | 設計での対応 | 対応状況 | 備考 |
|---------------------------------------|-------------|----------|------|
| **DispatchProxy制約（staticメソッド非対応）** | TraceHelper提供 | ✅ 対応済み | 手動トレースAPIで解決 |
| **new Thread でのコンテキスト非伝播** | TraceContext.Restore | ✅ 対応済み | 明示的コンテキスト復元 |
| **Fire-and-Forgetでの親終了問題** | StartLinkedTrace | ✅ 対応済み | Link機能で関連付け |
| **Parallel.ForEachでの親指定必要** | ParallelTraceHelper | ✅ 対応済み | 専用ヘルパー提供 |
| **機密情報漏洩リスク** | SensitiveParameters | ⚠️ Phase 2 | 自動マスクは後回し |
| **リフレクションオーバーヘッド** | MethodTraceInfoCache | ✅ 対応済み | キャッシュで軽減 |
| **JSONシリアライズコスト** | MaxDepth制限 | ✅ 対応済み | 深度5で制限 |
| **トレースデータ量** | サンプリング機能 | ⚠️ Phase 2 | SamplingRate設定予定 |

### 2.2 対応状況サマリー

|---------|---------|---------|--------|
| 技術的制約 | 4 | 0 | 0 |
